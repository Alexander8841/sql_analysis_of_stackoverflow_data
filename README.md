# SQL Analysis of StackOverflow Data

## Project Goal
Extract insights from StackOverflow database using SQL queries and advanced data analysis techniques.

---

## Data Description
Relational database containing:
- posts
- users
- tags
- activity metrics

---

## Objectives
- Write complex SQL queries
- Analyze user activity
- Extract meaningful patterns from relational data

---

## Tools & Technologies
- SQL (joins, aggregations, subqueries)
- Window functions
- PostgreSQL (or similar)
- Python (optional for visualization)

---

## What Was Done
- Queried relational database using SQL
- Used joins to combine multiple tables
- Applied window functions for advanced analysis
- Calculated metrics such as:
  - user activity
  - post popularity
  - engagement

---

## Key Results
- Identified most active users and tags
- Extracted patterns in user behavior
- Built reusable analytical SQL queries
